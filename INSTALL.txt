Accounting Management

Windows:
1. Run AccountingManagement-Setup-1.0.4.exe.
2. Open the app on the server PC first.
3. Keep the Windows app open while Android users connect.
4. The installer is per-user and should not need "Run as administrator".

Android:
1. Install AccountingManagement-Android-v1.0.4.apk.
2. Open Settings.
3. Set Server URL to the PC address shown in the Windows app, for example:
   http://192.168.1.10:4181
4. Use the update button in the top bar when a GitHub release is available.

Database:
- The app uses price_offer.db.
- Backups can be created from Settings.
- The server PC must stay on the same network as Android devices.

Default server port:
4181
